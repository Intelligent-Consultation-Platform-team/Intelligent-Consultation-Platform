## 七、AI 问诊接口文档（建议稿）

> 说明：以下接口文档用于前后端联调初稿，优先满足“患者发起 AI 问诊、持续对话、保存记录、生成建议、查询历史”的核心闭环。接口风格建议采用 RESTful 风格，返回结构统一，便于前端接入与后续扩展。

### 1. 设计目标
- 支持患者以文本方式发起 AI 问诊
- 支持多轮上下文对话
- 支持问诊记录保存、恢复与历史查询
- 支持 AI 生成初步建议、风险提示与就医建议
- 支持后续扩展语音、图片、检验单等多模态能力

### 2. 通用约定

#### 2.1 基础信息
- **接口前缀**：`/api/ai-consultation`
- **请求方式**：`GET` / `POST`
- **数据格式**：`application/json`
- **认证方式**：`Authorization: Bearer <token>`
- **时间格式**：`YYYY-MM-DD HH:mm:ss`

#### 2.2 统一响应结构
```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": 1710000000000
}
```

#### 2.3 通用状态码
- `200`：成功
- `400`：参数错误
- `401`：未登录或 token 失效
- `403`：无权限
- `404`：资源不存在
- `500`：服务器内部错误

#### 2.4 核心对象说明

**AI 问诊会话 consultationSession**
- `sessionId`：会话 ID
- `patientId`：患者 ID
- `patientName`：患者姓名
- `status`：会话状态（`active` / `closed` / `archived`）
- `chiefComplaint`：主诉
- `createdAt`：创建时间
- `updatedAt`：更新时间

**消息 consultationMessage**
- `messageId`：消息 ID
- `sessionId`：会话 ID
- `role`：消息角色（`user` / `assistant` / `system`）
- `content`：消息内容
- `contentType`：内容类型（`text` / `image` / `file`）
- `createdAt`：创建时间

---

### 3. 接口清单

#### 3.1 创建 AI 问诊会话
- **URL**：`POST /api/ai-consultation/session`
- **说明**：患者发起一次新的 AI 问诊会话

**请求参数**
```json
{
  "chiefComplaint": "头痛、发热两天",
  "symptomTags": ["头痛", "发热"],
  "age": 28,
  "gender": "female"
}
```

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "sessionId": "cs_202501010001",
    "status": "active",
    "createdAt": "2025-01-01 10:00:00"
  },
  "timestamp": 1710000000000
}
```

#### 3.2 发送问诊消息
- **URL**：`POST /api/ai-consultation/message`
- **说明**：向指定会话发送用户消息，并获取 AI 回复

**请求参数**
```json
{
  "sessionId": "cs_202501010001",
  "content": "我最近咳嗽，有点胸闷，需要注意什么？",
  "contentType": "text"
}
```

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "userMessage": {
      "messageId": "msg_u_001",
      "role": "user",
      "content": "我最近咳嗽，有点胸闷，需要注意什么？",
      "createdAt": "2025-01-01 10:01:00"
    },
    "assistantMessage": {
      "messageId": "msg_a_001",
      "role": "assistant",
      "content": "根据你描述的症状，建议先观察是否伴有发热、气促或持续加重...",
      "createdAt": "2025-01-01 10:01:02"
    },
    "riskLevel": "medium",
    "suggestion": "如症状持续或加重，请及时前往呼吸科就诊。"
  },
  "timestamp": 1710000000000
}
```

#### 3.3 获取会话消息列表
- **URL**：`GET /api/ai-consultation/session/{sessionId}/messages`
- **说明**：查询某次 AI 问诊的历史消息

**请求参数**
- `sessionId`：路径参数，会话 ID
- `page`：页码，默认 `1`
- `pageSize`：每页条数，默认 `20`

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "messageId": "msg_u_001",
        "role": "user",
        "content": "我最近咳嗽，有点胸闷，需要注意什么？",
        "contentType": "text",
        "createdAt": "2025-01-01 10:01:00"
      },
      {
        "messageId": "msg_a_001",
        "role": "assistant",
        "content": "根据你描述的症状，建议先观察是否伴有发热、气促...",
        "contentType": "text",
        "createdAt": "2025-01-01 10:01:02"
      }
    ],
    "total": 2,
    "page": 1,
    "pageSize": 20
  },
  "timestamp": 1710000000000
}
```

#### 3.4 获取会话详情
- **URL**：`GET /api/ai-consultation/session/{sessionId}`
- **说明**：查询会话基础信息与摘要

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "sessionId": "cs_202501010001",
    "patientId": 10001,
    "patientName": "张三",
    "status": "active",
    "chiefComplaint": "头痛、发热两天",
    "summary": "患者主诉头痛、发热两天，伴轻微乏力",
    "riskLevel": "medium",
    "createdAt": "2025-01-01 10:00:00",
    "updatedAt": "2025-01-01 10:01:02"
  },
  "timestamp": 1710000000000
}
```

#### 3.5 结束会话
- **URL**：`POST /api/ai-consultation/session/{sessionId}/close`
- **说明**：主动结束一次问诊会话

**请求参数**
```json
{
  "closeReason": "用户已获取建议，结束问诊"
}
```

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "sessionId": "cs_202501010001",
    "status": "closed"
  },
  "timestamp": 1710000000000
}
```

#### 3.6 查询我的 AI 问诊历史
- **URL**：`GET /api/ai-consultation/session/my`
- **说明**：分页查询当前登录用户的问诊历史列表

**请求参数**
- `page`：页码
- `pageSize`：每页条数
- `status`：可选，会话状态

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "sessionId": "cs_202501010001",
        "chiefComplaint": "头痛、发热两天",
        "riskLevel": "medium",
        "status": "closed",
        "createdAt": "2025-01-01 10:00:00"
      }
    ],
    "total": 1,
    "page": 1,
    "pageSize": 10
  },
  "timestamp": 1710000000000
}
```

### 4. 建议扩展接口

- **URL**：`GET /api/ai-consultation/template`
- **说明**：返回常见问诊引导语、提示词模板、免责声明模板

#### 4.2 上传图片资料
- **URL**：`POST /api/ai-consultation/upload`
- **说明**：上传症状图片、报告单等附件，返回文件 URL

#### 4.3 风险分级结果查询
- **URL**：`GET /api/ai-consultation/session/{sessionId}/risk`
- **说明**：查询当前会话的风险等级、推荐科室与建议
